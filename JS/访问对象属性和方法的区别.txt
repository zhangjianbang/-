 属性是一个变量，表示对象的特征，如颜色，大小等。
 方法是一个函数，表示对象的操作，如奔跑，跳跃等。

---》 取属性的方法有两种：“.”，“[ ]”
var obj={name:"zjb",eat=function(){console.log("吃东西")}}
obj.name="zjb"，obj.eat="吃东西";
obj["name"]="zjb"

！！“.”能取对象的属性和方法，“[ ]”只能取属性，不能取方法。

“.”和"[ ]"的优缺点
1.语法方面：
	+点表示法的对象属性名是标识符，后者的属性名是一个字符串。

2.灵活性：
	+ 对象中可以有多个属性或方法，用“.”运算法存取一个对象的属性时，其属性名是用标识符来表示，在Js中，表示符需要被逐字输入，它不是一种数据类型，因而程序不能对其操作。标识符是静态的，程序必须对其硬解码。当我们从某个文本框或者一串字符串中取出字符串时，因为有数据类型，所以不能作为点运算的标识符来访问对象的属性和方法。
	+ 当使用数组[ ]来存取对象属性时，属性名是用字符串表示，字符串是js中的基本数据类型，因而可以进行操作。

3.性能方面：
	+ 数组[ ]在存取属性时会进行表达式运算，而“.”是直接存取。
	+ 当动态存取属性时只能用数组表示法，而不能用点表示法。
	+ 点表示法一般用作静态存储属性，数组表示法用作动态存储属性。

Example：编写一个函数，接受两个参数，obj = {a:{b:[1,2,3]}}，path = ["a","b",2]，根据path路径输出对象的具体值。
	+ 思路，因为路径是数组，属性名是字符，所以要用到数组表示法来读取属性。
	
	function test(obj,path){
		var i=0;
		while(i<path.length){
			obj = obj[path[i]];
			i++;
		}
		return obj;
	}
